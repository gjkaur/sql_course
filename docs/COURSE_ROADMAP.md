# Course Roadmap

## Progression

```
Module 1 (Beginner)     → Foundation: schema, constraints, basic queries
    ↓
Module 2 (Intermediate)  → Design: normalization, indexes, EXPLAIN
    ↓
Module 3 (Intermediate) → Queries: subqueries, JOINs, optimization
    ↓
Module 4 (Advanced)     → Security: ACID, locks, RBAC, backup
    ↓
Module 5 (Advanced)    → Integration: Python, procedures, triggers
    ↓
Module 6 (Advanced)    → JSON: JSONB, hybrid schema
    ↓
Module 7 (DBA)         → Tuning: indexes, partitioning, monitoring
    ↓
Capstone               → Full project: e-commerce system
```

## Estimated Timeline

- **Full-time**: 6-8 weeks
- **Part-time (10 hrs/week)**: 12-16 weeks

## Dependencies

- Module 1 required for all
- Module 2 recommended before 3 (for index concepts)
- Module 4 before 5 (for transaction context)
- Modules 5-7 can be done in parallel after 4
